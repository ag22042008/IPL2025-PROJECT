IPL 2025 Match & Bowling Analysis

This project performs an exploratory data analysis (EDA) on IPL 2025 match and bowling data to uncover insights related to team strategies, bowling efficiency, workload management, and match impact. The analysis combines match-level trends with player-level bowling performance to reflect real-world cricket decision-making.

Analysis of the relationship between economy rate, wickets, and overs bowled.

Evaluation of whether high wicket takers are also match-winning bowlers using 4W and 5W hauls.

Team-wise comparison of bowling reliability and efficiency.
